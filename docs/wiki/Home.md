# NVR Viewer Wiki

Guides for people running NVR Viewer to watch and share recordings from their own Dahua NVR/DVR — not developer docs (see `docs/` in the repo root for those).

- [Getting Started](Getting-Started.md) — finding your NVR's IP/credentials, opening ports, first login
- [Configuration](Configuration.md) — every environment variable, explained
- [Troubleshooting](Troubleshooting.md) — black screen, expired share links, connection errors, H.265/H.264 confusion

For bugs or feature requests, open a GitHub issue. For security issues, see [SECURITY.md](../../SECURITY.md) in the repo root — please don't file those as public issues.
